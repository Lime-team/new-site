## How to clone
### 1. clone by url / ssh / github cli
### 2. use ' pip install -r requirements.txt 'to install pakages
### 3. flask-migrate:
flask db init

flask db migrate -m "table"

flask db upgrade
### 4. run app.py: ' python app.py '
